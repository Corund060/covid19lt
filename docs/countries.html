COVID-19 Europos šalių statistika

Atnaujinta: 2021-01-23 12:27:54.

Pagal 2020-05-27 Lietuvos vyriausybės nutarimą į Lietuvą be karantino galima atvykti Europos ekonominės erdvės valstybių, Šveicarijos, Didžiosios Britanijos ir Airijos piliečiams ir jose teisėtai gyvenantiems žmonėms, jeigu toje valstybėje, sergamumas koronaviruso infekcija (COVID-19) per pastarąsias 14 kalendorinių dienų neviršijo 25 nustatytų koronaviruso atvejų 100 tūkst. gyventojų.

Čia pateikiamas sąrašas Europos šalių ir atvejų skaičius 100 tūkst gyventojų nuo 2021-01-09 iki 2021-01-22.

x  %>% filter(date == max(date)-days(1) | date == max(date) - days(14)) %>% group_by(administrative_area_level_1) %>% 
    summarize(cases14 = round(diff(confirmed)/unique(population)*1e5,2)) %>% 
    arrange(cases14) %>% rename(country = administrative_area_level_1) %>% mutate(rank = 1:n()) %>% select(country, cases14) %>% datatable( 
         extensions = c('FixedColumns',"FixedHeader"),
          options = list(scrollX = TRUE, 
                         paging=FALSE,
                         fixedHeader=TRUE))

Kaimyninių šalių palyginimas

Šiame grafike pavaizduotas atvejų skaičius per paskutines 14 dienų, 100 tūkst. žmonių. Paskutinės dienos duomenys gali būti netikslūs. Duomenys imami iš Covid-19 Data Hub. Lietuvos duomenys ten atsilieka diena ir ateina su vėlavimais.
